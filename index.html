Porque usar o Contador de Caracteres e Palavras?

Apesar de simples, o Contador de Caracteres tem várias aplicações, sendo útil para quem deseja tornar mais prática a produção de textos para o Twitter (cujo limite é de 140 caracteres), campanhas de links patrocinados (35 caracteres para o título do anúncio, 70 para o texto e 35 para a URL de destino), ou até as meta-tags do SEO (a otimização de sites, que tem 63 caracteres no title e 160 no description). Utilizar um Contador de Caracteres e Palavras permite gerenciar melhor estes espaços que possuem número limitado de caracteres.

Como usar o contador?

Basta digitar ou colar o texto que deseja no campo abaixo:

Caracteres: 0 Palavras: 0 Linhas: 0
